- What is Forest? @{2024-10-07}
	- Forest is a note-taking/brainstorming interface made with Textual, built around the idea of organizing notes into nested branches and being able to zoom into branches @{2024-10-07}
	- To exit Forest, press Ctrl-Q @{2026-02-16}
- Navigation @{2026-02-16}
	- Up and down arrows to move between notes @{2026-02-16}
	- Left arrow zooms out (moves up in the tree), right arrow zooms in (focuses on the current branch) @{2026-02-16}
	+ Space toggles collapse/expand on any note with children @{2026-02-16}
		- Look! This branch is collapsed! @{2026-02-16}
			- It can have so many things inside @{2026-02-16}
	- 0-9 jumps to a bookmarked note @{2026-02-16}
- Creating notes @{2026-02-16}
	- Press Enter to add a new note below the cursor @{2026-02-16}
	- While typing, you can press Tab or Shift+Tab to indent or de-indent it @{2026-02-16}
- Editing notes @{2024-01-22}
	- Press Backspace to edit a note. Press Enter when done, or Esc to cancel @{2026-02-16}
	- x toggles a note as complete (dims the note and its children) #DONE @{2026-02-16}
	- X toggles hiding all #DONE notes from view @{2026-02-16}
	- Tab and Shift+Tab indent and de-indent a note @{2026-02-16}
	- Delete removes a note @{2026-02-16}
	- u and d move a note up or down among its siblings @{2026-02-16}
	- h cycles through highlight colours #HL1 @{2026-02-16}
		- primary highlight #HL1 @{2026-02-16}
		- secondary #HL2 @{2026-02-16}
		- tertiary #HL3 @{2026-02-16}
	- z to undo, Z to redo #HL3 @{2026-02-16}
	- y yanks (copies) the note text to the system clipboard @{2026-05-04}
	- Y yanks the note and all its descendants to the system clipboard (useful when terminal text-selection is awkward) @{2026-05-04}
	- Typing { NOW } (without the spaces) in a note will be replaced with the current timestamp @{2026-02-16}
- Saving @{2026-02-16}
	- Press s to save. If you have unsaved changes, "save" will appear in the top bar @{2026-02-16}
	- Auto-save is also enabled by default @{2026-02-16}
- Command mode (press : to enter) @{2026-02-16}
	- :?<query> searches within the current context (regex), :?*<query> searches the whole tree (regex) @{2026-02-16}
		- Use up/down arrows to cycle through matches, Enter to accept, Esc to cancel @{2026-02-16}
		- if ">" appears in the search query, searching will compare against path strings, not just the note string @{2026-02-16}
		- An empty query (:? or :?*) shows similar notes to current note in the side panel @{2026-02-16}
	- :sn opens the sticky note board, showing #HL3 notes in the current context. :sn* searches the whole tree @{2026-02-24}
		- :sn <filter> filters by highlight or regex, e.g. :sn #HL1, :sn* #HL2, :sn todo @{2026-02-24}
		- :snr recovers the last sticky note board at the same cursor position (⮺ in the status bar indicates a board is recoverable) @{2026-03-05}
	- :b or :bookmark toggles a bookmark on the current note @{2026-02-16}
		- Press ` to open the side panel and see existing bookmarks @{2026-02-16}
		- Jump to a bookmark by pressing its number key (0-9) @{2026-02-16}
		+ A maximum of 10 bookmarks are supported @{2026-02-16}
			- if all slots are filled in, a new bookmark will replace the least recently used bookmark @{2026-02-17}
	- :j+ <text> adds a timestamped journal entry (a journal branch is auto-created, organized by year and month) @{2026-02-16}
	+ :run [<index>] runs ! shell commands or follows [[PATH]] references @{2026-02-16}
		- e.g. run the :run command on this note [[SAVING > AUTO-SAVE IS ALSO]] @{2026-02-16}
		- If multiple [[PATH]] references exist, use :run 0, :run 1, etc. @{2026-02-16}
	- :random jumps to a random note in the current context, :random* jumps to a random note in the whole tree @{2026-03-06}
	- :insert <name> inserts a predefined template subtree @{2026-02-16}
	- :archive set tags the cursor note with #ARCHIVE, permanently hiding it (and its children) from view while still saving it to disk @{2026-04-17}
		- :archive unset removes the #ARCHIVE tag @{2026-04-17}
		- :archive show reveals archived branches for the session (an Ⓐ appears in the status bar); :archive hide returns to the default hidden state @{2026-04-17}
		- archived notes are excluded from search, random jump, and sticky notes as long as :archive hide is in effect @{2026-04-17}
	- :help shows the help panel @{2026-02-16}
- Expiring notes (reminders and timers) @{2026-07-08}
	- Give a note a countdown by tagging it: type the # symbol, then T-, then a duration such as 7d, 30m, 2mo, or 1y @{2026-07-08}
	- Press r on an expired note to renew it (restart its countdown for the same duration) @{2026-07-08}
	- To make a note restart itself automatically each time it expires (a repeating reminder), add a * right after the hyphen -- for example a T-*1d tag repeats every day @{2026-07-08}
	- If a note's text starts with ! (a shell command note), the command runs automatically when the note expires. Combine it with the repeating form for a periodic job, e.g. a !backup.sh note set to repeat daily @{2026-07-08}
- Side panel @{2026-02-16}
	- Press ` to cycle through the side panel views (bookmarks, perpetual journal) or hide it @{2026-02-16}
- Copying and Pasting @{2026-04-09}
	- Press "c" to add the cursor note to the top of the copied list. Press "c" again to remove it. @{2026-04-09}
	- Press "v" to paste the top of the copied list below the cursor note @{2026-04-09}
	- Press "C" to cycle the paste target (rotates the copied list without moving the cursor) @{2026-04-09}
	- Press "V" to move to the next copied note (also rotates the list, so the visited note becomes the new paste target) @{2026-04-09}
	- Press "l" to insert a [[path]] link to the top copied note (removes it from the list); follow with :run @{2026-04-22}
- What's up with the coloured strip on the left? @{2026-02-16}
	- This strip reflects the age of a note. New notes are bright, and as they get older they slowly fade @{2026-02-16}
- Flashcards @{2026-03-05}
	- To treat a note as a two-sided flashcard in the sticky-note view, you simply need to include "::" @{2026-03-05}
	- This is side A :: This is side B @{2026-03-05}
	- This is also side A :: @{2026-03-05}
		- And the first child is used for side B @{2026-03-05}
	- Use ":sn ::" to generate a sticky note view of flash cards @{2026-03-05}
- Doodling @{2026-04-22}
	- If configured (add `"doodle_pane_visible": true` to config.json), a panel will appear on the side that you can draw in. @{2026-04-22}
	- Click inside the panel to focus, and then start drawing! Use Shift-M to switch modes. Use Esc or click out to switch focus back. @{2026-04-22}
	- Doodles stack like backlit sheets -- doodles for the current context are more visible, with doodles from ancestor nodes being visible, but less so @{2026-04-22}
	- Use ":doodle show|hide|clear" to show the panel, hide it, or clear the doodle for the current context @{2026-04-22}
